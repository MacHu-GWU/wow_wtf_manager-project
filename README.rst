Install
------------------------------------------------------------------------------

``wow_wtf_manager`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install wow_wtf_manager

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade wow_wtf_manager
